Usage:

```
imgcat 0.1.0

USAGE:
    imgcat [FLAGS] [OPTIONS] <path>

FLAGS:
    -h, --help                     Prints help information
        --inline
        --preserve-aspect-ratio
    -V, --version                  Prints version information

OPTIONS:
        --height <height>     [default: auto]
        --width <width>       [default: auto]

ARGS:
    <path>
```

> cargo run -- me.jpg

### install:

```
cargo install --path .
```
